Darwin is an open-source playground that enables you to build serverless bioinformatics workflows using the PythonSDK and other frameworks such as Nextflow.
It takes top teams months to build infrastructure with the compute, storage, and user-friendly interface necessary to run bioinformatics pipelines at scale.
